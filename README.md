# Rakuten Grid


## Sizes Default
##### $screen-container: 1200px;
##### $screen-desktop: 980px;
##### $screen-mobile: 979px;


## Simple Syntax

### col-d-
Usage for desktop

### col-m-
Usage for mobile


```bash
<div class="rk-container">
    <div class="rk-row">
        <div class="col-m-12 col-d-6">
            Your code here
        </div>

        <div class="col-m-12 col-d-6">
            Your code here
        </div>
    </div>
</div>

```
